# 📊 PhonePe Business Analytics Dashboard | Power BI

## 📌 Overview

This project is a comprehensive Business Analytics Dashboard built in **Microsoft Power BI** using a PhonePe transaction dataset. The objective is to transform raw transactional data into meaningful business insights through interactive dashboards, KPI tracking, and advanced DAX calculations.

The dashboard enables stakeholders to monitor business performance, identify revenue trends, analyze customer behavior, and make data-driven decisions using an intuitive and recruiter-ready visualization interface.

---

## 🚀 Features

- Data Cleaning using Power Query
- Data Modeling with Relationships
- Custom Date Table
- Measures Table for organized DAX calculations
- Interactive KPI Cards
- Month-over-Month (MoM) Analysis
- Year-over-Year (YoY) Analysis
- Revenue Trend Analysis
- Service-wise Revenue Analysis
- Customer Segmentation
- Weekday vs Weekend Performance Analysis
- Age Group Analysis
- Interactive Tooltips
- Dynamic Filters & Slicers
- Business Storytelling Dashboard
- Professional Dashboard Design

## 📂 Project Workflow

### 1. Data Collection
- Imported PhonePe business dataset from Excel.

### 2. Data Cleaning
- Removed duplicates
- Handled missing values
- Corrected data types
- Standardized categorical values

### 3. Data Modeling
- Created relationships between tables
- Built a dedicated Date Table
- Implemented star-schema style modeling

### 4. DAX Development
Created measures for:

- Revenue
- Transactions
- Growth %
- Month-over-Month Analysis
- Year-over-Year Analysis
- Average Metrics
- Dynamic KPIs

### 5. Dashboard Design

Developed an interactive dashboard including:

- KPI Cards
- Revenue Trend Charts
- Category Analysis
- Customer Insights
- Tooltips
- Slicers
- Interactive Navigation

### 6. Business Insights

The dashboard provides insights into:
